class qf_lib.backtesting.contract.contract.Contract(symbol: str, security_type: str, exchange: str, contract_size=1)[source]¶

Bases: object

Represent a single contract.

Parameters

  • symbol (str) – symbol of the asset handled later on by the Broker (e.g. “MSFT”)

  • security_type (str) – e.g. ‘STK’ for a stock

  • exchange (str) – exchange on which the asset should be traded

  • contract_size (int) – size of the contract (default 1)
